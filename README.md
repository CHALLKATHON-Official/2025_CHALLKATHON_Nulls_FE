# ⏳ 인생 시계 (Life Clock)

> "내 인생 몇 % 살았지?"  
당신의 생일과 기대 수명을 기반으로 인생의 진행률을 시각화해주는 웹 애플리케이션입니다.  
타임라인 기능을 통해 주요 인생 이벤트를 기록하고, 친구와 비교해보며 삶의 궤적을 돌아볼 수 있어요.
